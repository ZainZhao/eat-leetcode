- 计算倒序数值

```java
class Solution {
    public boolean isPalindrome(int x) {
        if(x < 0)return false;
            
        int cur = 0;
        int num = x;
        while(num != 0) {
            cur = cur * 10 + num % 10;
            num /= 10;
        }
        return cur == x;
    }
}

```

- 计算倒序数值（优化）

```java
class Solution {
    public boolean isPalindrome(int x) {
 		if (x < 0 || (x % 10 == 0 && x != 0)) 
            return false;
        
        int revertedNumber = 0;
        while (x > revertedNumber) {
            revertedNumber = revertedNumber * 10 + x % 10;
            x /= 10;
        }
        return x == revertedNumber || x == revertedNumber / 10;
    }
}
```
