# 448 找到所有数组中消失的数字

```markdown
Label：数组

给定一个范围在  1 ≤ a[i] ≤ n ( n = 数组大小 ) 的 整型数组，数组中的元素一些出现了两次，另一些只出现一次。

找到所有在 [1, n] 范围之间没有出现在数组中的数字。

    输入:
    [4,3,2,7,8,2,3,1]

    输出:
    [5,6]
```

- map存储，然后遍历
```java
class Solution {
    public List<Integer> findDisappearedNumbers(int[] nums) {
        Map<Integer,Integer> map = new HashMap<>();
        for (int i = 0; i < nums.length; i++) {
            map.put(nums[i], i);
        }
        // 遍历判断哪些没有存在
        List<Integer> list = new ArrayList<>();
        for (int i = 1; i <= nums.length; i++) {
            if (!map.containsKey(i)) {
                list.add(i);
            }
        }
        return list;
    }
}
```

- 双重循环

```java
class Solution {
    public List<Integer> findDisappearedNumbers(int[] nums) {
        Arrays.sort(nums);
        // 遍历判断哪些没有存在
        List<Integer> list = new ArrayList<>();
        for (int i = 1; i <= nums.length; i++) {
            int flag = 0;
            for (int j = 0; j < nums.length; j++) {
                if (nums[j] == i) {
                    flag = 1;
                    break;
                }
            }
            if (flag == 0) {  // 遍历一次没有找到
                list.add(i);
            }
        }
        return list;
    }
}
```

- 原地修改数组

```java
class Solution {
    public List<Integer> findDisappearedNumbers(int[] nums) {

        List<Integer> res = new ArrayList<>();
        int i = 0;
        while (i < nums.length) {
            if (nums[i] == i + 1) {
                i++;
                continue;
            }

            int idealIdx = nums[i] - 1;   // 当前位置出现的数应该正确出现的位置
            if (nums[i] == nums[idealIdx]) {
                i++;
                continue;
            }

            int tmp = nums[i];
            nums[i] = nums[idealIdx];
            nums[idealIdx] = tmp;
        }

        for (int j = 0; j < nums.length; j++) {
            if (nums[j] != j + 1) {
                res.add(j + 1);
            }
        }
        return res;
    }
}
```

- 范围之外

```java
class Solution {
    public List<Integer> findDisappearedNumbers(int[] nums) {
        int n = nums.length;
        for (int num : nums) {
            int x = (num - 1) % n; // 每个出现过的元素 + n
            nums[x] += n;
        }
        List<Integer> ret = new ArrayList<Integer>();
        for (int i = 0; i < n; i++) {
            if (nums[i] <= n) {  // 没有出现过的
                ret.add(i + 1);
            }
        }
        return ret;
    }
}
```
