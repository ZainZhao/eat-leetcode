## 冒泡排序

时间复杂度：O（$n^2$​​）

时间复杂度：O（$1$）

> 每一次迭代选出第i大的值 

```java
public class BubbleSort {
    public int[] bubbleSort(int[] A, int n) {
        for (int i = 1; i < n; i++) { // 控制交换次数
            for (int j = 0; j < n - i; j++) {  // 注意index
                if (A[j] > A[j + 1]) {
                    int temp = A[j];
                    A[j] = A[j + 1];
                    A[j + 1] = temp;
                }
            }
        }
        return A;
    }
}
```
